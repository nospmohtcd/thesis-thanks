# 🎓 Thesis Thanks
**A shareable tribute for the people who make research possible.**

"You’ve done the hard work. You’ve finished the research, survived the write-up, and dedicated a page to the people who carried you through. But unless they hunt down your library deposit, they’ll likely never see it."

### Features
* **Metadata Fetching:** Auto-fill thesis titles using DOI or citation strings.
* **Smart Tagging:** Categorize thanks (Family, Supervisor, etc.).
* **Google Scholar Support:** Link directly to repository clusters.
* **Email Integration:** Generate pre-filled "Thank You" emails with one click.

### Tech Stack
* **Frontend:** Streamlit
* **Backend:** Google Sheets (via st-gsheets-connection)
